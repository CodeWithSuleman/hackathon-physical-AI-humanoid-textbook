# Data Model: RAG Agent with OpenAI Agents SDK + FastAPI

This document describes the key data entities for the RAG Agent feature.

## Query Request

The input object for the `/query` endpoint.

**Attributes**:
- `question`: (string, required) The natural language question from the user.

## Agent Response

The output object from the `/query` endpoint.

**Attributes**:
- `answer`: (string) The coherent answer generated by the agent.
- `references`: (list of Reference) A list of source references.

## Reference

An object representing a source document used in the answer.

**Attributes**:
- `url`: (string) The URL of the source document.
- `heading`: (string) A relevant heading or title from the source.
